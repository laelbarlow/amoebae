module_amoebae_constrain_mb module¶

Contains functions for adding contstraint commands to MrBayes input files.

module_amoebae_constrain_mb.constrain_mb_with_tree(alignment, tree, out_alignment=None)[source]¶

Take a nexus alignment and newick tree and write a new alignment file with MrBayes constraint commands added to constrain the tree search to only those topologies that match input tree topology (at all nodes).

module_amoebae_constrain_mb.get_taxon_number_dict(alignment)[source]¶

Take a nexus alignment file and return a dictionary with taxon names as keys and their numbers as values.

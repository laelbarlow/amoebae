module_mask_nex module¶

Mask a nex alignment by adding a new taxon, “MASK” sequence.

This is only a rough mask that is applied based on very simple criteria.

It would be good to eventually add more sophistocated criteria, especially criteria based on similarity (according to a scoring matrix) rather than just identity, or number of sequences without gaps.

module_mask_nex.apply_mask_criteria(column)[source]¶

Apply simple masking criteria to a single column, and return ‘-‘ if the column does not meet the criteria, and ‘I’ if it does.

module_mask_nex.mask_alignment(alignment)[source]¶

Takes an alignment object and adds a ‘MASK’ sequence using certain criteria for inclusion of positions.

module_mask_nex.mask_nex(infilepath, outfilepath=None)[source]¶

Takes a filepath and adds a MASK sequence.

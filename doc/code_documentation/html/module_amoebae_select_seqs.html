module_amoebae_select_seqs.optimize_sequence_selection(file_with_seqs, model_name, essential_taxa)[source]¶

Add/remove homologous sequences from a dataset to optimize for long stem lengths and low average branch lengths for clades of interest.

The essential_taxa argument is a list of names corresponding to taxonomic groups of eukaryotes (such as “Amorphea” or “Diaphoretickes”) which appear in the genome info csv file in the genomes directory.

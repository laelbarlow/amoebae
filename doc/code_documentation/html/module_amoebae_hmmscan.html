module_amoebae_hmmscan module¶

Module for amoebae script with functions for running hmmscan (from the HMMer3 software package).

module_amoebae_hmmscan.all_nex_to_hmm(indirpath)[source]¶

Converts all nex files in a given directory to .hmm (writes .hmm files).

module_amoebae_hmmscan.call_hmmpress(infilepath)[source]¶

Calls hmmpress on a file to make a concatenation of .hmm files into an HMM database for searching with hmmscan.

module_amoebae_hmmscan.cat_hmm(hmm_paths, outfilepath)[source]¶

Takes a list of .hmm file paths, and outputs a file that is a concatenation of these, and returns its path.

module_amoebae_hmmscan.determine_if_is_hmmdb(infp)[source]¶

Return True if the given file is an HMM database (generated using hmmpress from the HMMer3 software package), and return False otherwise.

module_amoebae_hmmscan.get_hmm_paths(indirpath)[source]¶

Returns a list of all .hmm files in a given directory.

module_amoebae_hmmscan.make_hmm_db(indirpath, hmmdbname=None, outdirpath=None)[source]¶

Takes .hmm files in a given input directory, and outputs a searchable HMM database (files) in a given output directory.

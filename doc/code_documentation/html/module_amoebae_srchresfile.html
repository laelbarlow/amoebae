Module for defining a module for parsing similarity search output files with various formats.

Parses relevant information as needed from similarity search output file, which may be one of several types.

module_amoebae_srchresfile.get_hmmer_hit_seq_coord(searchio_hit_obj, db_file, extra=0)[source]¶

Takes a Bio.SearchIO hmmer3-text hit object and returns specially formatted coordinates for the subsequence (top HSP) that match the query.

Note: The coordinates are the literal start and end residue numbers for the relevant portion of the subject sequence.

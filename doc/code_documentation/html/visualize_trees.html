visualize_trees module¶

Module for visualizing phylogenetic trees (parsing newick trees and writing as formatted pdf files with images).

visualize_trees.add_borders_to_pdf_with_latex(in_tree_file, out_tree_file)[source]¶

Make a latex file, add the existing pdf as an image, and write to pdf, so that there are borders.

visualize_trees.customize_node_styles_for_clades_to_remove(t, highlight_for_removal)[source]¶

Take an ete3 tree object and a list of nodes with paralogues, and modify the node styles to emphasize important features.

visualize_trees.customize_node_styles_for_paralogue_clades(t, nodes_with_paralogues)[source]¶

Take an ete3 tree object and a list of nodes with paralogues, and modify the node styles to emphasize important features.

visualize_trees.customize_node_styles_for_visualization(t)[source]¶

Take an ete3 tree object , and modify the node styles for better visualization.

visualize_trees.determine_whether_first_subtrees_same_sp_overlap(i, j, overlapping_species)[source]¶

Take two TreeNode objects and determine whether the first object (i) has a top-level subtree that has the same overlap as the two nodes do.

visualize_trees.get_branch_support_from_node_name(node_name)[source]¶

Parse node name of internal nodes resulting from parsing a newick tree with ete3 with the format=1 option (branch supports become node names for internal nodes). Return a single float corresponding to the bootstrap proportion.

visualize_trees.get_first_leaf_from_species(tree_obj, species_start_string)[source]¶

Take an ete3 TreeNode object and return a node object with a name that starts with a given string.

visualize_trees.get_first_subtrees(t)[source]¶

Take an ete3 TreeNode object and return the clades on either side of the root node.

visualize_trees.get_largest_subtree(t)[source]¶

Take an ete3 TreeNode object and return the largest subtree.

visualize_trees.get_nodes_with_paralogues(t)[source]¶

Find nodes that contain paralogues, and return as a list.

visualize_trees.get_subs_model_from_iqtree_file(file_with_subs_model_name)[source]¶

Take the input directory path, and return the substitution model used (listed in the output.iqtree file).

visualize_trees.get_subs_model_from_mb_file(file_with_subs_model_name)[source]¶

Take the input directory path, and return the substitution model used (listed in the .nex file).

visualize_trees.label_pdf(in_tree_file, out_tree_file, method, subs_model, timestamp)[source]¶

Taka pdf file of a phylogenetic tree and label with a given substitution model and timestamp.

visualize_trees.lighten_color(color, amount=0.5)[source]¶

Takes a hex colour code and returns a hex code for the same colour but lighter by multiplying (1-luminosity) by the given amount (fraction).

Take a list of ete3 TreeNode objects and return only those that are not subtrees of others in the list.

visualize_trees.translate_int_node_names_to_support(t)[source]¶

Parse branch support values out of internal node names, add one of them to each node as branch support (a single float), and delete the internal node names.

visualize_trees.translate_int_node_support_to_prob(t)[source]¶

Parse branch support values and change them to probabilities instead of perentages.

visualize_trees.visualize_tree(method, timestamp, taxa_to_root_on, highlight_paralogues, highlight_for_removal, tree_file, tablefilename, typeseqsfilename, add_clade_names, file_with_subs_model_name)[source]¶

Take relevant files and write PDF files with tree figures, and return output file paths.

visualize_trees.visualize_tree_in_dir(indp, method, timestamp, taxa_to_root_on, highlight_paralogues, add_clade_names_from_file)[source]¶

Given a path to a directory with output files from a phylogenetic analysis, write PDF files with images of the results, and return the paths to those files.

indp is the path to the directory with results. method is either ‘iqtree’, ‘mrbayes’, or ‘raxml’. taxa_to_root_on is a list of names such as ‘Klebsormidium’ or ‘Klebsormidium nitens’

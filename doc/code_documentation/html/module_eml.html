module_eml module¶

For sending emails from scripts. Some code taken from http://naelshiab.com/tutorial-send-email-python/

This works for sending emails from gmail accounts for which “less secure apps” have been enabled in the security settings.

module_eml.send_eml(subject, body)[source]¶

Sends an email using the from and to addresses and password defined in the settings.py file.

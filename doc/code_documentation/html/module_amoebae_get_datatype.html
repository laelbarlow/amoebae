module_amoebae_get_datatype module¶

Module defining function(s) for inferring the data type from a give file.

module_amoebae_get_datatype.get_datatype_for_sequence_string(concat_seq)[source]¶

Take a sequence string and return ‘prot’ or ‘nucl’ depending on the content of letters from the alphabet.

module_amoebae_get_datatype.get_dbtype(f)[source]¶

Opens input fasta file, examines the first 10 records, and returns either ‘prot’ or ‘nucl’ depending on what type the sequences are.

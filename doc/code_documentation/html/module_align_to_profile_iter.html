module_align_to_profile_iter module¶

Script for iteratively aligning sequences from a fasta file one-by-one with an existing alignment using MUSCLE’s -profile option.

****** Ideally, the fastas should be added to the alignment in decreasing order of how well they align to the HMM (based on score).

Usage:
align_to_profile_iter.py <nex starting alignment> <fasta file with seqs to align to nex alignment>
module_align_to_profile_iter.align_iteratively(inalignfilepath, temp_subdirname, outfilepath)[source]¶

Takes an afa file and aligns it one-by-one to sequences in files in the temporary subdirectory, and outputs the final alignment in fasta format.

Do the whole process

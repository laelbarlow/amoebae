module_afa_to_nex module¶

This module is for afa_to_nex.py script and others, which use the same functions.

module_afa_to_nex.afa_to_nex(infilepath, outfilepath=None)[source]¶

Takes an aligned fasta file (protein) and writes a nexus format file.

module_afa_to_nex.delete_extra_mesquite_lines(infilepath)[source]¶

Remove extra blocks added by Mesquite, if present. This prevents Biopython from raising an error about an unmatched ‘end’ in one of the nexus blocks. The extra blocks are not necessary anyway.

module_afa_to_nex.determine_alphabet(filepath)[source]¶

Automatically determine what alphabet to use for a given file.

module_afa_to_nex.nex_to_afa(infilepath, outfilepath)[source]¶

Takes a nexus file (protein) and writes an aligned fasta file.

module_afa_to_nex.nex_to_mbnex(infilepath, outfilepath, mrbayescodeblocks)[source]¶

Takes a nexus format alignment file and converts to a format for use in a phylogenetic analysis using MrBayes.

module_afa_to_nex.nex_to_phylip(infilepath, outfilepath)[source]¶

Takes nexus format file and makes a phylip format file. Warning: Taxon names must be no more than 10 alphanumeric characters.

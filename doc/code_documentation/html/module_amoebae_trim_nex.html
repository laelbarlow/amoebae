module_amoebae_trim_nex module¶

Trim an nex alignment using the last taxon “MASK” sequence as a guide.

Usage:
trim_nex.py <path to nex alignment with MASK>
module_amoebae_trim_nex.trim_alignment(alignment)[source]¶

Recursively trims an alignment by removing positions with ‘I’ in the MASK sequence.

module_amoebae_trim_nex.trim_nex(infilepath, outfilename)[source]¶

Takes a filepath for a nexus alignment, and writes a new file containing a trimmed alignment.

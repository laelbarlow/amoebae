module_boots_on_mb module¶

Module that defines functions for mapping bootstrap values onto a MrBayes tree topology.

module_boots_on_mb.combine_supports(boot_newick, prob_newick, combined_figtree_newick)[source]¶

Takes two newick files and writes another newick file that when opened in figtree will show posterior probabilities and bootstrap values together.

***Should add a function to check whether the two trees being compared are rooted on the same node, because that might affect the accuracy of comparisons.

module_boots_on_mb.contre_to_newick(intreepath, outtreepath)[source]¶

Takes a .con.tre file output from MrBayes and writes the same tree in newick format.

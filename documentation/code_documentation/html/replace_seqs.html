replace_seqs module¶

Module for amoebae script with functions for the replace_seqs command.

replace_seqs.afa_to_fa(infilepath, outfilepath)[source]¶

Takes an aligned fasta file (protein) and writes a .fa file.

replace_seqs.replace_seqs_in_alignment_with_seqs_from_fasta(alignment, fasta=None)[source]¶

Takes sequences from an alignment, finds the top blastp hit in a given fasta file for each one, then aligns the top hits to the alignment, then removes the original sequences from the alignment, writes the alignemnt, and returns the path to the output alignment file.

Note: Might be better to explicitly mask the new sequences to only the columns that align with the original trimmed sequences instead of applying general masking criteria, because some input alignments might be masked more stringently.

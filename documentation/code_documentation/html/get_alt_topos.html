get_alt_topos module¶

Contains functions for finding all relevant alternative topologies for a given tree.

get_alt_topos.determine_if_polytomy(subtree)[source]¶

Determine if the given subtree (represented as a set, which may contain sets) is a polytomy or not.

get_alt_topos.get_all_alt_topologies(subtrees)[source]¶

Take a list of ete3 TreeNode objects and return a list of TreeNode objects representing all possible backbone topologies for relationships between the input clades.

get_alt_topos.get_all_unique_unrooted_bifurcating_topologies_for_n_taxa(n)[source]¶

Given an integer from 3 to 7, generate all unique unrooted bifurcating tree topologies for that given number of taxa.

get_alt_topos.get_leaf_names_to_represent_n_taxa(n)[source]¶

Take n integer and return a list of the same number of unique strings that can be easily replaced with other taxon names or subtrees.

get_alt_topos.get_newick_tree_string_from_set(s)[source]¶
get_alt_topos.get_partitions_of_set(s)[source]¶

Recursively partition elements in an input set.

get_alt_topos.get_polytomy_for_treenode(treenode)[source]¶

Take an ETE3 TreeNode object and return a TreeNode object with all the same leaf nodes, but just as a polytomy.

get_alt_topos.get_total_num_topos_for_n_taxa(n)[source]¶

Given a number of taxa (integer greater than 2), return an integer representing the theoretical total number of possible topologies (without polytomies) for relationships between those taxa.

get_alt_topos.trees(leaves, polytomies=False)[source]¶

Generate all alternative trees (in newick format) for a given list of leaf names, by recursively partitioning the input set of leaf names in every possible way. By default polytomies are excluded, but can be included by setting the polytomies option to True.

module_amoebae_name_replace module¶

Module defining functions for coding and decoding taxon/sequence names in alignments and tree files for input to phylogenetics software. Based on the name_replace.pl script written by Jed Barlow.

module_amoebae_name_replace.codenames_nex(infp, aamodelpr=None)[source]¶

Takes a path to a multiple sequence alignment file in nexus format and writes a new file with coded names.

module_amoebae_name_replace.generate_code_name(number)[source]¶

Take the number of the sequence in the input file, and output an alphabetical code that uniquely corresponds to that number.

module_amoebae_name_replace.get_conversion_dict_from_table(tablefile)[source]¶

Take a conversion table and return a dictionary for uncoding names.

module_amoebae_name_replace.write_afa_with_code_names(infile, outfile, tablefile)[source]¶

Take an aligned (or unaligned) fasta file, and write a fasta file with coded names, and a table for converting names back.

module_amoebae_name_replace.write_newick_tree_with_coded_names(infile, outfile, tablefile, quoted_names=False)[source]¶

Take a newick file and replace full taxon names with coded names from a table.

module_amoebae_name_replace.write_newick_tree_with_uncoded_names(infile, outfile, tablefile, quoted_names=False)[source]¶

Take a text file with a newick tree, and write a new one with names converted back to original names (given values in a conversion table), and optionally put quotation marks around the names in the output tree.

module_amoebae_name_replace.write_nex_with_code_names(infile, outfile, tablefile)[source]¶

Take a nexus file, and write a nexus file with coded names, and a table for converting names back.
